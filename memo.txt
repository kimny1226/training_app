# Linux エイリアス登録
$ vi ~/.bashrc
alias ll='ls -al'
alias vssh='vagrant ssh -c "cd /vagrant; bash"';
alias art='php artisan'

$ source ~/.bashrc


1. GitHubでリポジトリ作成
2. clone 必要に応じてブランチ切替(develop等)
$ git clone https://github.com/kimny1226/training_app.git

3. laravel6 インストール
$ cd training_app/
$ composer create-project --prefer-dist laravel/laravel app 6.*

4. add、commit、push

5. ライブラリ取得
$ cd app/
$ composer install

5. 認証(公式より) https://readouble.com/laravel/6.x/ja/authentication.html
$ composer require laravel/ui "^1.0" --dev
$ art migrate

6. nodeモジュール css生成(bootstrap有効)
$ npm install && npm run dev



## Service作成
1. 本体作成
app\app\Services\XxxService.php
2. ServiceProvider作成
$ art make:provider XxxServiceProvider
3. Facade作成
app\app\Facades\XxxService.php
4. alias登録
app\config\app.php
5. 利用
use XxxService;
XxxService::xxxxx();
